Abstract

As embodied navigation agents gain prevalence in the physical world, assessing their adversarial robustness against physical adversarial examples is crucial, yet remains unexplored. This work aims to bridge this gap by developing physically realizable adversarial patches for navigation scenarios. However, existing patch generation methods, which typically attach directly to object surfaces, lack practicality and are limited to 2D pixel interference, failing to address multi-view visual perception challenges in embodied navigation. To overcome these limitations, this paper proposes a new framework called AdvNav for generating highly deployable adversarial patches. Specifically, for physical practicality, the adversarial patches are designed not to touch the target objects directly, such as appearing as a painting on a wall. For effectiveness across multiple views, the patches' texture are optimized through an iterative process using differentiable rendering, effectively suppressing target detection from any critical viewing angle. Experiments on the Object Goal Navigation (ObjectNav) task involved sampling navigation trajectories within a scene and generating tailored patches for various targets. The results demonstrated a significant reduction in navigation success rates by 50%. Furthermore, black-box tests across 20 additional scenarios and 5 navigation models confirmed the patches' robust transferability. For further details and results, please refer to our project page at https://AdvNav/.
